# Titanic Survival Prediction

## Project Overview
This project is part of a Data Science Internship task.
The objective is to build a simple predictive model using Scikit-learn
to predict passenger survival on the Titanic dataset.

## Dataset
- Source: Seaborn Titanic Dataset
- Target Variable: survived (0 = No, 1 = Yes)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Jupyter Notebook

## Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation

## Model Performance
- Algorithm: Logistic Regression
- Accuracy: ~79%
